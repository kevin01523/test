<pre><code>python local.py -s &lt;server_ip&gt; \
                -p &lt;port&gt; \
                -k &lt;keyphrase&gt; \
                -m &lt;encryption&gt; \
                -o &lt;obfus&gt; \
                -O &lt;protocol&gt; \
                -l &lt;local_port&gt;

<p>Check logs:</p>
<pre><code>tail -f /var/log/shadowsocks.log
</code></pre>
<p><code>-h</code> shows the documentation.</p>
